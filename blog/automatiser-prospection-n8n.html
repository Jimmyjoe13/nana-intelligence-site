Workflow n8n : Automatisation Complète de la Prospection

Voici le **workflow que j’utilise pour mes clients** (et pour moi) :

  1. Scraping LinkedIn → Apify ou PhantomBuster extrait les profils (ex: "CTO SaaS France")
  2. Enrichissement des données → Hunter.io vérifie l’email, Clearbit enrichit le profil
  3. Filtrage intelligent → Seulement les entreprises > 10 employés, postes pertinents
  4. Génération de message personnalisé → L'IA rédige un email unique (pas de template)
  5. Envoi via Resend ou Gmail → Pas de ban, haute délivrabilité
  6. Relance LinkedIn (J+3) → Message court + like du dernier post
  7. Qualification automatique → Si le lead répond, il est tagué "Chaud" dans le CRM
  8. Intégration HubSpot / Airtable → Tout est centralisé, pas de perte de données

Résultat ? 100+ leads qualifiés par mois, sans lever le petit doigt.

3 Pièges à Éviter (Sinon, tu te fais ban)

n8n est puissant. Mais si tu abuses, tu perds ton accès.

1. Scraping LinkedIn trop agressif

LinkedIn détecte les scrapings. Solution :
- Limite à 100 requêtes/jour
- Utilise Apify ou PhantomBuster (rotations d’IP)
- Ne scrape pas en continu

2. Envoi d’emails en masse

100 emails/jour ? OK. 1000 ? Spam.
Solution :
- Utilise Resend ou Brevo (délivrabilité optimisée)
- Limite à 50-100 emails/jour par domaine
- Ajoute des délais aléatoires entre les envois

3. Pas de personnalisation

Un email "Cher {prénom}" ? C’est du spam.
Solution :
- Utilise un modèle LLM pour générer un message unique
- Mentionne le dernier post LinkedIn, la tech stack, un client similaire
- Ton taux de réponse passe de 5% à 30%
